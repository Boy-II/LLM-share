# 關注技術清單

## 1. Google ADK (Agent Development Kit)

### 簡介
Google 的 Agent Development Kit (ADK) 是一個靈活且模組化的框架,用於開發和部署 AI agents。雖然針對 Gemini 和 Google 生態系統進行了優化,但它也支援其他 AI 模型。

### 主要特色
- **自主執行單元**: Agent 是自包含的執行單元,能夠自主行動以實現特定目標
- **工具整合**: 支援多種工具類型(內聯函數、資料庫、第三方 API 等)讓 agent 與外界互動
- **模組化設計**: 提供靈活的架構,便於開發複雜的 AI agent 系統
- **MCP Toolbox 整合**: 可與 Model Context Protocol (MCP) 工具箱整合

### 應用場景
- 研究型 agent 開發
- 旅行助理等任務型 agent
- 需要多工具協作的複雜任務

### 參考資源
- 官方文檔: https://google.github.io/adk-docs/
- Google Cloud 部落格介紹

## 2. TopView.ai

### 簡介
TopView.ai 是一個基於 AI 的行銷影片生成平台,標榜為「#1 Marketing Video Agent」。它能將產品圖片轉換成病毒式行銷影片,大幅簡化影片製作流程。

### 主要特色
- **AI 影片生成**: 只需上傳產品圖片或短影片,即可自動生成專業行銷影片
- **病毒式內容庫**: 擁有超過 1000 萬個高表現影片的訓練數據
- **AI Avatar**: 可將任何照片轉換為虛擬角色,並讓其展示產品
- **多平台優化**: 專為 TikTok、Instagram、YouTube Shorts、Facebook 等平台優化
- **多語言支援**: 支援 30+ 種語言,自動調整字幕和配音

### 效能優勢
- **10 倍**更快的影片製作速度(從數週縮短至數分鐘)
- **80-90%** 成本節省
- **零學習曲線**: 任何人都能立即開始創作

### 適用對象
- 電商賣家
- DTC 品牌
- 聯盟行銷人員
- 行銷代理商
- 內容創作者

### 核心功能
1. **Viral Hooks**: 創造吸睛的影片開場
2. **Product Anyshoot**: AI 生成產品拍攝場景
3. **Character Swap**: 替換影片中的人物角色
4. **Avatar Marketing Video**: 使用 UGC 風格的 Avatar 生成行銷影片

### 參考資源
- 官方網站: https://www.topview.ai/

## 3. OpenAI Agent Kit / Swarm

### 簡介
OpenAI 推出的 agent 開發工具集,包含 **AgentKit**(完整的企業級工具集)和 **Swarm**(輕量級多 agent 協作框架)。

### Swarm Framework
- **輕量級框架**: 專注於讓 agent 協調和執行變得輕量、高度可控且易於測試
- **多 agent 系統**: 用於構建和編排多個 AI agents 的協作系統
- **實驗性質**: 目前仍是實驗性框架,主要用於教育和探索

### AgentKit
- **完整工具集**: 為開發者和企業提供完整的工具,用於建構、部署和優化 agents
- **企業就緒**: 支援生產環境的部署需求

### 主要特色
- **Agent 互動循環**: 管理 agent 互動、函數調用和潛在的任務交接
- **清晰的控制**: 保持系統的清晰性和可控性
- **複雜對話流程**: 支援複雜的對話流程和任務分配

### 應用場景
- 多 agent 協作系統
- 企業級 AI 助理
- 複雜的任務自動化

### 參考資源
- GitHub: https://github.com/openai/swarm
- 官方公告: https://openai.com/index/introducing-agentkit/
