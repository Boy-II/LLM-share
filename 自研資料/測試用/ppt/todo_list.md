# 待辦事項與討論目標

## 專案目標
- 協助生成 PowerPoint 簡報的圖像提示詞
- 根據報告內容進行分頁規劃
- 為每一頁生成符合需求的圖像描述

## 待辦事項
- [ ] 接收並理解報告完整內容
- [ ] 規劃簡報分頁結構
- [ ] 為每一頁生成對應的圖像提示詞
- [ ] 確認提示詞符合 16:9 規格及 PowerPoint 使用需求

## 討論記錄

### 專案需求確認
- **內容形式**：逐頁說明，透過口說報告提供
- **我的任務**：截取段落重點並生成提示詞
- **分頁結構**：無特定結構，頁數不限
- **視覺風格**：盡可能統一，但議題大方向改變時可調整
- **生成工具**：Nano Banana Pro
- **提示詞格式**：依範例格式（主題、色調、文字位置、視覺元素、主標、副標、規格）

### 工作流程
1. 接收每頁報告內容
2. 截取重點並提煉主要訊息
3. 依內容生成對應的圖像提示詞
4. 記錄於 prompt.md
